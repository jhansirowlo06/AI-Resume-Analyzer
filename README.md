🚀 Project Overview

AI Resume Analyzer & Builder is a full-stack web application developed using the MERN stack and Groq AI to help job seekers improve their resumes. The application analyzes uploaded resumes, calculates ATS compatibility, compares resumes with job descriptions, provides AI-powered suggestions for improvement, and enables users to build and download professional resumes.

The platform includes an interactive Resume Builder with live preview, AI-generated resume enhancement, ATS score prediction, job description matching, resume history management, and PDF generation. It is fully deployed using Vercel for the frontend and Render for the backend, with MongoDB Atlas as the database.

🎯 Project Objective

The primary objective of this project is to help students and professionals optimize their resumes for Applicant Tracking Systems (ATS), receive AI-generated improvement suggestions, and create professional resumes in a simple and user-friendly interface.

This version is concise, professional, and suitable for GitHub, LinkedIn, and your resume.
